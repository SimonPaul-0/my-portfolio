CreditDNA

Privacy-first, India-focused credit literacy web app — local scoring flows, statement OCR in the browser, and a “For Banks” partner surface.

Problem Explain credit in plain language without sending sensitive documents to a random server in the prototype phase.

What I did Implemented a client-side SPA (hash routing, local demo accounts), questionnaire-based scoring, PDF/image OCR via Tesseract.js + PDF.js, heuristic parsing, Three.js DNA helix hero, and documented API-style examples for lenders.

Stack Single-file app (index.html), Tailwind CDN, Three.js, Tesseract.js, Vercel.

Result Coherent product narrative from landing → score → OCR → partner docs, all in public source.
